Descriptive statistics

Created a table to summarize statistic for purple and red urchins by finding mean, median, standard deviation and sample size.

urchins_summary <- urchins_data %>% 
  group_by(common_name) %>% 
  summarize(mean = mean(size_cm),
            median = median(size_cm),
            sd = sd(size_cm),
            sample_size = n()) %>% 
  kable(col.names = c("Urchin name",
                      "Mean (cm)",
                      "Median (cm)",
                      "Standard Deviation",
                      "Sample Size"),
        digits = 1) %>% 
  kable_minimal()

Takeaways: The purple urchin population has a lower central tendency (4.5cm mean, 4.5cm median) than the red urchin population (6.8cm mean, 7.0cm median). The purple urchins are more concentrated around the central tendency, with a smaller spread and lower standard deviation (1.2) while red urchins are more spread with a higher standard deviation (1.9).

Section 2

Explore the purple urchin data

Compared purple urchin counts by site and year. Done by filter() only purple urchins, changing date class to “Date”,and creating new columns.

purple_urchins <- urchins_data %>% 
  filter(common_name == "Purple Urchin") %>% 
  mutate(date_new = mdy(date)) %>% 
  mutate(year_new = year(date_new)) %>% 
  mutate(site_full = case_when(
    site == "NAPL" ~ "Naples Reef",
    site == "AQUE" ~ "Arroyo Quemado",
    site == "CARP" ~ "Carpinteria Reef",
    site == "MOHK" ~ "Mohawk Reef",
    site == "IVEE" ~ "Isla Vista Reef"
  ))

Descriptive statistics

Created a table to store of total number of purple urchins observed by site and year.

total_purple_site_year <- purple_urchins %>%
  group_by(site_full, year_new) %>% 
  summarize(sample_size = n())
